# Task API 🚀

API REST desenvolvida com Python e FastAPI.

## Tecnologias

- Python
- FastAPI
- REST API

## Funcionalidades

✅ Criar tarefas  
✅ Listar tarefas  
✅ Atualizar tarefas  
✅ Deletar tarefas  


## Executar projeto

Instalar dependências:

pip install -r requirements.txt


Executar:

uvicorn main:app --reload


Documentação:

http://127.0.0.1:8000/docs
